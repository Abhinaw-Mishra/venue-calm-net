# 🚀 Rapid Crisis – Smart Emergency Response Platform

## 📌 Problem

In emergency situations, people often face delays in getting help due to lack of real-time communication and coordination.

## 💡 Solution

Rapid Crisis is a web platform that allows users to instantly report emergencies, share their location, and get faster assistance using smart automation.

## 🔥 Key Features

* 🚨 One-click emergency alert
* 📍 Real-time location sharing
* ⚡ Fast response system
* 🧭 Nearby help visibility
* 📊 Simple dashboard

## 🛠 Tech Stack

* Frontend: React + Vite
* Styling: Tailwind CSS
* Deployment: Vercel / Lovable
